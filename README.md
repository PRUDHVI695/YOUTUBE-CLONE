# YouTube Clone 🎬

A responsive YouTube Clone built using HTML, CSS, and JavaScript.  
This project recreates the basic user interface of YouTube with a clean and modern design.

## 🚀 Features
- Responsive Design
- YouTube-like Homepage
- Video Thumbnail Cards
- Sidebar Navigation
- Search Bar Interface
- Modern UI Layout

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
```bash
YOUTUBE-CLONE/
│
├── index.html
├── style.css
├── script.js
├── assets/
└── README.md
```

## 🎯 Purpose of the Project
This project was created to:
- Practice Frontend Development
- Improve HTML, CSS, and JavaScript skills
- Learn Responsive Web Design
- Build Real-World UI Projects

## 🚀 Getting Started

### Clone the Repository
```bash
git clone https://github.com/PRUDHVI695/YOUTUBE-CLONE.git
```

### Run the Project
Open `index.html` in your browser.
